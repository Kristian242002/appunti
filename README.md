# Repository Appunti – Università e Studio Personale

Questa repository contiene una raccolta di appunti personali relativi al mio percorso di studi e approfondimenti individuali.

Gli argomenti presenti non riguardano necessariamente tutte le materie universitarie, ma includono ciò che ritengo utile, fondamentale o meritevole di essere ricordato a lungo termine.

## Contenuto

All’interno si possono trovare:

* appunti legati a corsi universitari
* concetti teorici fondamentali
* note su linguaggi di programmazione studiati in autonomia (ad esempio Dart o altri linguaggi non presenti nel piano di studi)
* schemi, riassunti, concetti chiave e spiegazioni personali

Questi appunti non vogliono essere un sostituto dei corsi ufficiali, ma un archivio personale per memorizzare e consolidare ciò che studio.

## Obiettivo

Mantenere in un unico luogo ciò che imparo e ciò che reputo necessario rivedere nel tempo, migliorando la memorizzazione a lungo termine e avendo un archivio digitale sempre accessibile.

## Modifiche e contributi

Chiunque può scaricare i file (inclusi quelli LaTeX) e modificarli liberamente secondo le proprie esigenze, oppure correggere eventuali errori presenti.

Pull request e suggerimenti sono ben accetti.

## Note finali

La repository non rappresenta un corso completo né una guida ufficiale, ma un punto di riferimento personale per approfondimenti e ripasso.
